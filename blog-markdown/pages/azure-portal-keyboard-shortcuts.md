# Keyboard shortcuts in the Azure portal

## Actions
* C-N Create a new resource
* G+B Open All services
* ALT+Shift+Up Arrow Move up the selected left sidebar item

## Go to
* G+D Go to Dashboard
* G+A Go to All resources
* G+R Go to Resource Group
